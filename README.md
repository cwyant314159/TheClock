### Building

This project uses CMake as the build system. A `debug` and `release` preset is
provided to simplify building.

```bash
cmake --preset debug
cmake --build --preset debug

./build/debug/source/c-clock/c-clock   # C variant
./build/debug/source/c++clock/c++clock # C++ variant
```
